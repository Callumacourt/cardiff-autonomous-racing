# Process Explained
Here is described which script contains the guide step by step for the required development in the simulator or annotating cones for the cone detection algorithm.

## CONE_ANNOTATION_SETUP.md file
This file describes the procedure to install and run the cone annotation tool which is used to create the database to train the cone detection algorithm

## SIMULATION_SETUP.md file
This file describes the procedure to install and run the simulation and autopilot system. Although is explicitly designed for the development of the autopilot system since it uses a packaged version of the simulator. This means that there cannot be any modifications to the simulator.

Follow this guide if you only intend to develop the autopilot system.

## SIMULATION_SETUP_DEV.md file
This file describes the procedure to install and run the simulation and autopilot system. This has the blueprints to modify the simulation in Unreal Engine 4.

Follow this guide if you intend to develop the simulation.
